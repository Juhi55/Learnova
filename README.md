# 📖 Learnova – AI Powered Study Assistant

![React](https://img.shields.io/badge/React-19-blue?logo=react)
![Node.js](https://img.shields.io/badge/Node.js-Express-green?logo=node.js)
![MongoDB](https://img.shields.io/badge/MongoDB-Atlas-brightgreen?logo=mongodb)


Learnova is an AI-powered study platform that transforms study materials into interactive learning resources. Users can upload PDF notes, generate AI summaries, quizzes, flashcards, chat with an AI tutor, and organize everything in a personal library.

---

# 🚀 Features

- 📄 Upload PDF notes
- 📝 AI Generated Summaries
- ❓ Automatic Quiz Generation
- 🗂 Flashcard Generation
- 🤖 AI Tutor Chat
- 📚 Personal Learning Library
- 🎯 Practice Mode
- 🔒 JWT Authentication
- 🌙 Dark & Light Theme
- 📱 Responsive UI
- ⚡ Modern Dashboard
- 🔍 Search & Navigation

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/Juhi55/Learnova.git
```

---

## Backend

```bash
cd backend
npm install
npm start
```

---

## Frontend

```bash
cd frontend
npm install
npm run dev
```
